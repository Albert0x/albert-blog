## 起点

写代码这么多年，**「技术沉淀」** 这件事一直没真正落地过。

每天忙着学习新框架、调代码、解决问题，下班想着「明天再总结」，结果就是一万次「明天」。

直到我意识到：**没有沉淀的能力是会蒸发的。**

## 这个博客要写什么

不是流水账，不是技术名词堆砌，就三个方向：

### 1. 工业互联网学习
- 工业系统集成的架构思考
- 能源管理 / 碳管理领域的技术探索
- 工业网关协议（Modbus、OPC UA、MQTT）的实践

### 2. AI 应用探索
- RAG 在知识库场景的落地尝试
- Agent 框架对比（LangChain / LlamaIndex / LangGraph）
- 大模型在领域场景的工程化思考

### 3. 全栈工程
- React 工程化最佳实践
- Python 后端开发心得
- 小程序 unibest 框架学习笔记

## 关于这里的内容

本博客所有文章为**个人学习实践记录**，技术方案与代码示例均为学习用途，不涉及任何商业项目细节。

## 为什么是现在

> 写博客最好的时间是十年前，其次是现在。

我已经错过了第一个最好时间，那就抓住第二个吧。
