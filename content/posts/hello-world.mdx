## 我是谁

我叫 Albert，目前在做工业互联网相关的软件开发，
主力栈是 **React + Python**，业余时间在自学 AI 应用方向（RAG、Agent 之类）。

工作日常包括：

- 给客户搭工业系统（涉及一些传感器、网关、协议）
- 写后台管理 / 数据看板 / 小程序
- 看大模型 / RAG 相关的资料，尝试小练手

## 为什么开这个博客

老实说不是为了流量，更不是为了「打造个人品牌」。
就一个很朴素的原因：

> **我学了不少东西，但很多学完没几天就模糊了。**

我希望通过**写下来**这个动作：

1. 逼自己用自己的话讲清楚（费曼学习法）
2. 留个痕迹，下次自己再翻能想起来
3. 万一有人看到对他也有点用，那是额外的好处

## 这里会写什么

主要是**学习笔记**，按方向分大概是：

- **工业互联网**：我在做什么、读了什么 spec、搞懂的概念
- **AI 应用**：RAG / Embedding / LangChain / 本地大模型 - 学到哪写到哪
- **全栈**：React / Next.js / TypeScript / Python 日常踩到的点
- **随笔**：偶尔的想法，比如这一篇

## 这里**不会**写什么

- 营销味很重的「3 大」「10 个坑」「实战指南」
- 我没真做过的项目
- 编造的数据（"召回率 91%" 这种）

写的东西我自己**也在学**，所以会有错。
看到的人发现错了直接告诉我，我会改。
